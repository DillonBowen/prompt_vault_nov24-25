# Prompt Vault

A glassmorphic, enterprise-grade React application for storing and searching prompts.

## Features

- **Glassmorphic UI**: Modern, aesthetic design with transparency and blur effects.
- **Search**: Real-time filtering by act, prompt content, or tags.
- **Copy to Clipboard**: One-click copying of prompts.
- **Responsive**: Works on desktop and mobile.
- **Data**: Pre-loaded with a comprehensive dataset of prompts.

## Setup

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- PapaParse (CSV parsing)
